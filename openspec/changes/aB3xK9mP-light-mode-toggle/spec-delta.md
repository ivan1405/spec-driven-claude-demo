# Spec Delta — Light Mode Toggle

## ADDED capabilities

- **light-mode** — A new capability. The application supports a light theme alongside the
  existing dark theme, switchable from a header control. Default is dark. No persistence.
  See `specs/light-mode/spec.md` for requirements and scenarios.

## MODIFIED capabilities

None. Existing capabilities (`task-badges`, `board-title`) are unaffected — priority
badge colors and the board title are not in scope for this change.

## REMOVED capabilities

None.
